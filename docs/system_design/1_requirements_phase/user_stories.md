# User Stories

## Authentication & Profile
- **US-001**: As a **New User**, I want to **register with my email**, so that I can access the platform.
- **US-002**: As a **User**, I want to **log in using Google**, so that I don't have to remember another password.
- **US-003**: As a **Security-Conscious User**, I want to **enable 2FA**, so that my account is protected.
- **US-004**: As a **User**, I want to **upload an avatar**, so that I can personalize my profile.

## Tenant Management
- **US-005**: As a **User**, I want to **create a new Tenant (Organization)**, so that I can collaborate with my team.
- **US-006**: As a **Tenant Owner**, I want to **invite colleagues by email**, so that they can join my workspace.
- **US-007**: As a **Invited User**, I want to **accept an invitation**, so that I can become a member of the tenant.
- **US-008**: As a **Tenant Owner**, I want to **remove a member**, so that I can manage access.

## Billing & Subscription
- **US-009**: As a **Tenant Owner**, I want to **subscribe to a Pro Plan**, so that I can unlock advanced features.
- **US-010**: As a **Tenant Owner**, I want to **update my payment method**, so that my subscription continues without interruption.
- **US-011**: As a **Tenant Owner**, I want to **view past invoices**, so that I can track expenses.

## API & Developer Experience
- **US-012**: As a **Developer**, I want to **generate an API token**, so that I can integrate third-party tools.
- **US-013**: As a **Developer**, I want to **view API documentation**, so that I understand how to consume the endpoints.
